HOW TO PUSH TO GITHUB
========================================

✅ Cleanup Complete:
   - All test files removed
   - All .bat files removed
   - All unnecessary .md files removed
   - Only essential files remain

========================================
PUSH COMMANDS
========================================

Run these commands in order:

1. Remove client's git folder:
   rmdir /s /q client\.git

2. Add all files:
   git add .

3. Commit:
   git commit -m "Complete Profit First D2C Analytics Platform"

4. Add remote:
   git remote add origin https://github.com/Aniket17200/Profitfirst.git

5. Set branch to main:
   git branch -M main

6. Push to GitHub:
   git push -u origin main

========================================
AFTER PUSHING
========================================

Your repository will be at:
https://github.com/Aniket17200/Profitfirst

To clone on another machine:
git clone https://github.com/Aniket17200/Profitfirst.git
cd Profitfirst
npm install
cd client && npm install && cd ..
cp .env.example .env
npm run dev
